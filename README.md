# 🎬 Netflix Data Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Netflix Movies and TV Shows dataset using Python. The objective is to clean the data, explore patterns, visualize trends, and derive meaningful insights about Netflix's content library.

This project was developed as part of a Data Analysis assignment and demonstrates fundamental data preprocessing, visualization, and analysis techniques.

---

## 📂 Dataset

* **Dataset:** Netflix Movies and TV Shows
* **Source:** Kaggle
* **Records:** Approximately 8,800+ titles
* **Features:** 12 columns including title, type, director, cast, country, release year, rating, duration, genre, and description.

---

## 🎯 Project Objectives

* Load and inspect the dataset.
* Clean missing and duplicate data.
* Perform Exploratory Data Analysis (EDA).
* Create multiple visualizations.
* Generate meaningful insights from the data.

---

## 🛠 Technologies Used

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Seaborn

---

## 📊 Exploratory Data Analysis

The project answers the following analytical questions:

1. Movies vs TV Shows distribution.
2. Top 10 countries producing Netflix content.
3. Distribution of content by release year.
4. Distribution of content ratings.
5. Most popular Netflix genres.

---

## 📈 Visualizations

The project includes multiple visualization types such as:

* Bar Chart
* Horizontal Bar Chart
* Line Chart
* Pie Chart
* Histogram
* Scatter Plot
* Heatmap

---

## 🔍 Key Insights

* Netflix contains significantly more Movies than TV Shows.
* The United States contributes the highest number of Netflix titles.
* Netflix experienced rapid content growth after 2015.
* TV-MA is the most common content rating.
* International Movies and Dramas are among the most popular genres.
